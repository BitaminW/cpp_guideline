## 포인터 정리

#### 이차원 배열 접근
  
```cpp
	int arr2[2][3] = { 1,2,3,4,5,6 };

	cout << arr2[0][3] << "\n";

```
다음과 같은 코드를 보면 arr2[0][3] 라는 코드가 있다. 인덱스를 넘겨서 넣었는데도 에러가 나지않는다 이는 다음과 같은 코드가 동일하기 떄문이다.

```cpp
	int arr[2][4] = { {1,2,3,4}, 
			 {5,6,7,8} };

	int arr2[2][3] = { 1,2,3,4,5,6 };
```

사실 2차원 배열을 설명하는 그림이라던지 작성을 할 떄 보기 쉽게 그리지만 정확히는 메모리가 일렬로 되어있기 때문이다.
그러므로 아래코드는 동일한 값이 나온다.

```cpp
	cout << arr2[0][3] << "\n";
	cout << *(arr2[0] + 3) << "\n";
	cout << *(arr2[1]) << "\n";
```

[출력결과]
```bash
4
```
